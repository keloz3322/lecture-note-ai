# Transcript Studio UI/UX Handoff

This folder contains the public, repo-safe UI/UX handoff for the next v0 Max design pass.

## Files

- `UIUX_Agent_Brief.md` - product positioning, information architecture, target screens, interaction requirements, visual system, and acceptance checklist.
- `V0_UIUX_ONESHOT_PROMPT.md` - the one-shot prompt intended for a new v0 Max session connected to this repository.

## Notes

- The source reference images are not committed because this repository is public and the images come from third-party design/reference sites.
- Use the source links in `UIUX_Agent_Brief.md` as visual references.
- The redesign should improve the existing working Transcript Studio app, not replace it with a static mock dashboard.
- Preserve current API routes, model selection behavior, Vercel environment usage, demo flows, and deployed functionality unless the prompt explicitly says otherwise.
